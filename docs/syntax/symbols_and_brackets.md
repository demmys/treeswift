### Symbols

```
Semicolon -> ';'

Colon -> ':'

Comma -> ','

Dot -> '.'

Arrow -> '->'

Hash -> '#'

Underscore -> '_'

Atmark -> '@'

Dollar -> '$'

BackQuote -> '`'

Minus -> '-'

DoubleQuote -> '"'

BackSlash -> '\'

PrefixAmpersand -> '&'

PrefixQuestion -> '?'
BinaryQuestion -> '?'
PostfixQuestion -> '?'

PostfixExclamation -> '!'

PrefixGraterThan -> '<'

PostfixLessThan -> '>'

BinaryDoubleEqual -> '=='

VariadicSymbol -> '...'
```

### Brackets

```
LeftParenthesis  -> '('
RightParenthesis -> ')'

LeftBrace  -> '{'
RightBrace -> '}'

LeftBracket ->  '['
RightBracket -> ']'
```
