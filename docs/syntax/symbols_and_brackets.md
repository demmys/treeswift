### Symbols

```
Arrow -> '->'

AssignmentOperator -> '='

Atmark -> '@'

BinaryQuestion -> '?'

Colon -> ':'

Comma -> ','

Dot -> '.'

PostfixExclamation -> '!'

PostfixLessThan -> '>'

PostfixQuestion -> '?'

PrefixAmpersand -> '&'

PrefixGraterThan -> '<'

PrefixQuestion -> '?'

Semicolon -> ';'

Underscore -> '_'

VariadicSymbol -> '...'
```

#### Symbols for literal parsing

```
BackQuote -> '`'

BackSlash -> '\'

Dollar -> '$'

DoubleQuote -> '"'

Minus -> '-'
```

### Brackets

```
LeftParenthesis  -> '('
RightParenthesis -> ')'

LeftBrace  -> '{'
RightBrace -> '}'

LeftBracket ->  '['
RightBracket -> ']'
```
