### Symbols

```
Semicolon -> ';'

Colon -> ':'

Comma -> ','

Dot -> '.'

Arrow -> '->'

Hash -> '#'

Underscore -> '_'

Atmark -> '@'

PrefixAmpersand -> '&'

PrefixQuestion -> '?'
BinaryQuestion -> '?'
PostfixQuestion -> '?'

PostfixExclamation -> '!'

Dollar -> '$'

PrefixGraterThan -> '<'

PostfixLessThan -> '>'

BinaryEqual -> '='

BinaryDoubleEqual -> '=='

PrefixHyphen -> '-'

VariableSymbol -> '...'
```

### Brackets

```
LeftParenthesis  -> '('
RightParenthesis -> ')'

LeftBrace  -> '{'
RightBrace -> '}'

LeftBracket ->  '['
RightBracket -> ']'
```
