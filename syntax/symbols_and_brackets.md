### Symbols

```
Semicolon -> ';'

Colon -> ':'

Comma -> ','

Arrow -> '->'

Hash -> '#'

Underscore -> '_'

Ampersand -> '&'

PrefixQuestion -> '?'
BinaryQuestion -> '?'
PostfixQuestion -> '?'

Dollar -> '$'
```

### Brackets

```
LeftParenthesis  -> '('
RightParenthesis -> ')'

LeftBrace  -> '{'
RightBrace -> '}'

LeftBracket ->  '['
RightBracket -> ']'
```
